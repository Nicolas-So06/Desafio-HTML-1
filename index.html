Historia WEB

A história da web é como uma grande jornada de transformação na maneira como nos comunicamos e usamos a tecnologia. Tudo começou lá em 1989, quando Tim Berners-Lee teve a ideia brilhante de criar a World Wide Web. Desde então, a web evoluiu muito! No início, as páginas eram bem simples, apenas textos e imagens paradas. Mas hoje, temos uma web cheia de aplicativos legais e interativos que tornam nossa experiência online muito mais divertida.

Clients & Servers

Quando falamos sobre programação web, existem dois conceitos importantes: "clients" e "servers". O client é como o navegador que usamos, como o Google Chrome ou o Safari. Ele nos permite acessar a internet e ver todas as coisas legais que estão disponíveis online. Já o server é como um grande armazém digital que guarda todas as informações, como páginas da web e arquivos. Sem os servidores, não poderíamos ver nada na internet.

Linguagem de Programação

Sobre as linguagens de programação, pense nelas como diferentes ferramentas que os desenvolvedores usam para criar coisas na web. Por exemplo, o HTML é usado para colocar o conteúdo na página, como textos e imagens. O CSS é usado para deixar a página bonita e estilosa, com cores e formatos. E o JavaScript é como o cérebro da página, dando vida a botões e animações legais. E existem muitas outras linguagens além dessas, como Python, PHP e Ruby, cada uma com suas próprias habilidades e usos especiais. Então, na próxima vez que você usar a internet, lembre-se de toda essa tecnologia incrível que está por trás dela.

Introdução do HTML

HTML, ou HyperText Markup Language, é a linguagem padrão para criar e estruturar páginas da web. Desenvolvido inicialmente por Tim Berners-Lee no início da década de 1990, o HTML desempenha um papel fundamental na construção de sites, permitindo a criação de conteúdo que pode ser interpretado e exibido pelos navegadores da web.

TAGS

  • html:

    Define o início e o fim do documento HTML.

  • head:

    Contém metadados e informações sobre o documento.

  • body:

    Contém o conteúdo visível da página.

  • h1, h2, h3, h4, h5 e h6:

    Cabeçalhos de diferentes níveis.

  • p:

    Define um parágrafo.

  • a:

    Define um hyperlink.

  • img:

    Insere uma imagem.

  • ul:

    Define uma lista não ordenada.

  • ol:

    Define uma lista ordenada.

  • li:

    Define um item de lista.

  • i:

    Deixa o texto itálico

  • strong:

    Deixa o texto em negrito

  • hr:

    Utilizada para inserir uma linha horizontal no conteúdo da página, fornecendo uma divisão visual entre seções ou elementos.

  • entre outros...

Atributos das TAGS

  • id

    Identifica exclusivamente um elemento na página.

  • href

    Define uma ou mais classes CSS para o elemento, permitindo estilização.

  • src

    Define o caminho de origem do 'img'

  • style:

    Permite a definição de estilos CSS para o elemento.

  • entre outros...

Na aula 3, avançamos na prática de desenvolvimento de um site, abordando duas novas funcionalidades: a tag hr e a criação de links âncora.

Além disso, aprendemos a criar links âncora, que são links que levam o usuário para uma parte específica da mesma página. Para isso, utilizamos o atributo id para identificar o elemento de destino e o atributo href com o valor #nome-do-id para criar o link.
